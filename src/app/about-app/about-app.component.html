About The Todo Application

ToDo is a task management app to help you stay organized and manage your day-to-day/future Goals. You can use this app to make task lists with time and work on acheiving those on time

How to Use the app

  1> Register if you are new user or else login.
  2> Create your todo list with targeted time.
  3> YOu can subscribe To Mail if you want mails in case of target not comple or new features added in app.

Below AWS Services used for Deployment and Notification

  Amazon Web Services are used to deploy the application:
  1> For DB > AWS RDS : To create mySql instance
  2> For UI > AWS S3 Buckets are used to deploy Angular code
  3> For Backend > AWS Elastic Beanstack is used to deploy or Spring boot Application
  4> For Email Subscription : AWS SNS is used
